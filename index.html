eclipse_install in WIN10

安装eclipse步骤存档

First,download and install jdk 1.7 version from The SUN official website. //for quick test you can press win+x and type instruction java -version to check if the jdk is installed successfully.

Then, set some system variables for java. This PC -> attribute -> advanced -> enviorment variables

*name JAVA_HOME

value C:\Program Files\Java\jdk1.7.0 /this value is where you install the jdk*/

*name CLASSPATH

*value.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

modify system variables

*name PATH

*value %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

Finally, activate your eclipse using your code.
